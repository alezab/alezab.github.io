    Wstęp - Podstawy hashowania

    Różnica między echo a echo -n

    # echo dodaje znak nowej linii (line feed)
echo "marek" | md5sum 
# Output: b0207a607303c08cd4f78c324952d7c2  -

echo "marek" | wc
#       1       1       6

# echo -n NIE dodaje znaku nowej linii
echo -n "marek" | wc
#       0       1       5

echo -n "marek" | md5sum 
# Output: e061c9aea5026301e7b3ff09e9aca2cf  -

    Ważne: Przy hashowaniu zawsze używaj echo -n aby uniknąć hashowania dodatkowego znaku nowej linii!

    Podstawowe komendy OpenSSL

    # Lista dostępnych algorytmów
openssl dgst -list 

# Przykładowe użycie
echo -n "marek" | openssl dgst -md5 
# Output: MD5(stdin)= e061c9aea5026301e7b3ff09e9aca2cf

    Zadanie 1.1 - Hash MD5

    Polecenie

    Pod adresem http://127.0.0.1:10001 działa prosty serwer HTTP udostępniający dwa endpointy: - http://127.0.0.1:10001/hash - pobieranie słowa do zahashowania - http://127.0.0.1:10001/submit - wysyłanie odpowiedzi

    Zadanie: Oblicz skrót MD5 podanego przez serwer losowego słowa i wyślij wynik w formacie hex.

    Rozwiązanie

    (a) Uruchomienie serwera

    docker run -p 10001:10001 docker.io/mazurkatarzyna/hashing-md5-ex1:latest

    (b) Pobranie słowa i obliczenie hash MD5

    # Pobierz słowo od serwera
curl -s 127.0.0.1:10001/hash | jq .

    {
  "session_id": "6224ec9042035711",
  "word": "lostfan"
}

    # Oblicz hash MD5
echo -n "lostfan" | openssl dgst -md5 
# Output: MD5(stdin)= b0ef584b7757e61150df570b16f110d5

    (c) Wysłanie odpowiedzi do serwera

    curl -s -X POST 127.0.0.1:10001/submit \
  -H "Content-Type: application/json" \
  -d '{"session_id": "6224ec9042035711", "hash_hex": "b0ef584b7757e61150df570b16f110d5"}' | jq .

    {
  "result": "Correct MD5 hash!"
}

    Zadanie 1.6 - Python: MD5

    Polecenie

    Napisz skrypt w języku Python, który wygeneruje hash MD5 dowolnego ciągu znaków podawanego jako argument wywołania skryptu. Sprawdź poprawność porównując z md5sum lub openssl. Wykorzystaj bibliotekę hashlib.

    Rozwiązanie

    Plik: zadanie_1_6.py

    #!/usr/bin/env python3
"""
Zadanie 1.6: Generowanie hash MD5 z wykorzystaniem hashlib
"""

import hashlib
import sys


def md5_hash_string(input_string: str) -> str:
    """
    Returns the MD5 hash of a given string.
    """
    # Konwertuj string na bajty
    input_bytes = input_string.encode("utf-8")
    # Stwórz obiekt MD5 hash
    md5_hash = hashlib.md5(input_bytes)
    # Zwróć hash w formacie hex
    return md5_hash.hexdigest()


if __name__ == "__main__":
    if len(sys.argv) > 1:
        text = sys.argv[1]
    else:
        text = "hello world"

    result = md5_hash_string(text)
    print(f"MD5 hash of '{text}': {result}")

    Testowanie

python3 zadanie_1_6.py "hello world"
# Output: MD5 hash of 'hello world': 5eb63bbbe01eeed093cb22bb8f5acdc3

# Weryfikacja z md5sum
echo -n "hello world" | md5sum
# Output: 5eb63bbbe01eeed093cb22bb8f5acdc3  -

    Zadanie 1.9 - Python: Bcrypt

    Polecenie

    Napisz skrypt w języku Python, który wygeneruje hash bcrypt dowolnego ciągu znaków podawanego jako argument wywołania skryptu. Sprawdź poprawność porównując z htpasswd. Wykorzystaj bibliotekę bcrypt.

    Rozwiązanie

    Instalacja biblioteki:

    pip install bcrypt

    Plik: zadanie_1_9.py

    #!/usr/bin/env python3
"""
Zadanie 1.9: Generowanie hash bcrypt z wykorzystaniem biblioteki bcrypt
Instalacja: pip install bcrypt
"""

import sys
import bcrypt


def bcrypt_hash_password(password: str) -> str:
    """
    Hash a password using bcrypt.
    """
    # Konwertuj hasło na bajty
    password_bytes = password.encode("utf-8")
    # Wygeneruj salt i hash hasła (10 rund)
    salt = bcrypt.gensalt(rounds=10)
    hashed = bcrypt.hashpw(password_bytes, salt)
    # Zwróć hash jako string
    return hashed.decode("utf-8")


def bcrypt_verify_password(password: str, hashed: str) -> bool:
    """
    Verify a password against a bcrypt hash.
    """
    # Konwertuj hasło i hash na bajty
    password_bytes = password.encode("utf-8")
    hashed_bytes = hashed.encode("utf-8")
    # Zweryfikuj hasło
    return bcrypt.checkpw(password_bytes, hashed_bytes)


# Example usage
if __name__ == "__main__":
    if len(sys.argv) > 1:
        pwd = sys.argv[1]
    else:
        pwd = "my_secure_password"

    # Hash the password
    hashed_pwd = bcrypt_hash_password(pwd)
    print(f"Password: {pwd}")
    print(f"Bcrypt hash: {hashed_pwd}")

    # Verify the password
    is_valid = bcrypt_verify_password(pwd, hashed_pwd)
    print(f"Verification: {is_valid}")

    # Test with wrong password
    is_valid_wrong = bcrypt_verify_password("wrong_password", hashed_pwd)
    print(f"Verification with wrong password: {is_valid_wrong}")

    Zadanie 1.10 i 1.12 - Identyfikacja: SHA-1

    Polecenie

    Mając dany początkowy ciąg znaków helloworld, który następnie został zahashowany, określ, jaka funkcja skrótu została wykorzystana do utworzenia hasha: 6adfb183a4a2c94a2f92dab5ade762a47889a5a1.

    Uwaga: Można użyć narzędzia hash-identifier lub hashid, ale możemy również napisać własne rozwiązanie w Python.

    Rozwiązanie

    Plik: zadanie_1_10_13.py

    #!/usr/bin/env python3
"""
Zadania 1.10-1.13: Identyfikacja typów hashów
Zamiast używać hash-identifier lub hashid, testujemy różne algorytmy
"""

import hashlib


def identify_hash(original_text: str, target_hash: str):
    """
    Próbuje zidentyfikować algorytm hash poprzez testowanie popularnych algorytmów
    """
    input_bytes = original_text.encode("utf-8")

    # Lista popularnych algorytmów hash z ich typową długością
    common_algorithms = {
        "md5": 32,
        "sha1": 40,
        "sha224": 56,
        "sha256": 64,
        "sha384": 96,
        "sha512": 128,
        "sha3_224": 56,
        "sha3_256": 64,
        "sha3_384": 96,
        "sha3_512": 128,
    }

    print(f"Tekst oryginalny: '{original_text}'")
    print(f"Hash do zidentyfikowania: {target_hash}")
    print(f"Długość hash: {len(target_hash)} znaków")

    # Sprawdź czy to bcrypt (zaczyna się od $2y$ lub $2a$ lub $2b$)
    if (
        target_hash.startswith("$2y$")
        or target_hash.startswith("$2a$")
        or target_hash.startswith("$2b$")
    ):
        print("\n✓ Zidentyfikowano: BCRYPT")
        print("  Format: $2y$rounds$salt+hash")
        return "bcrypt"

    print("\nTestowanie standardowych algorytmów hash:")
    print("-" * 60)

    found = False
    for algo_name, expected_len in common_algorithms.items():
        try:
            h = hashlib.new(algo_name, input_bytes)
            result = h.hexdigest()

            match = "✓ MATCH!" if result.lower() == target_hash.lower() else ""
            print(f"{algo_name:12} ({expected_len} znaków): {result[:20]}... {match}")

            if result.lower() == target_hash.lower():
                found = True
                print(f"\n✓ ZNALEZIONO! Algorytm: {algo_name.upper()}")
        except Exception:
            pass

    if not found:
        print("\nNie znaleziono pasującego algorytmu wśród standardowych hashów.")

    return None


if __name__ == "__main__":
    print("=" * 60)
    print("ZADANIE 1.10 i 1.12")
    print("=" * 60)
    identify_hash("helloworld", "6adfb183a4a2c94a2f92dab5ade762a47889a5a1")

    Zadanie 1.14 - Identyfikacja: RIPEMD-160

    Polecenie

    Mając dany początkowy ciąg znaków R3iSrSNmgU9SFHxVekUD, który następnie został zahashowany, określ, jaka funkcja skrótu została wykorzystana do utworzenia hasha: 48cab4b54bef42fddaa6353c68a20b369f40026e.

    Podpowiedź: Algorytm jest dostępny w bibliotece hashlib.

    Rozwiązanie

    Plik: zadanie_1_14.py

    #!/usr/bin/env python3
"""
Zadanie 1.14: Identyfikacja funkcji skrótu dla hashu 48cab4b54bef42fddaa6353c68a20b369f40026e
Tekst oryginalny: R3iSrSNmgU9SFHxVekUD
"""

import hashlib


def test_all_hashes(input_string: str, target_hash: str):
    """
    Testuje wszystkie dostępne algorytmy hash z hashlib
    """
    input_bytes = input_string.encode("utf-8")

    # Lista algorytmów dostępnych w hashlib
    algorithms = hashlib.algorithms_available

    print(f"Szukam algorytmu dla: '{input_string}'")
    print(f"Oczekiwany hash: {target_hash}")
    print(f"\nTestowanie {len(algorithms)} algorytmów...\n")

    found = False
    for algo_name in sorted(algorithms):
        try:
            # Stwórz hash dla danego algorytmu
            h = hashlib.new(algo_name, input_bytes)
            result = h.hexdigest()

            # Sprawdź czy hash się zgadza
            if result.lower() == target_hash.lower():
                print(f"✓ ZNALEZIONO! Algorytm: {algo_name}")
                print(f"  Hash: {result}")
                found = True
        except Exception:
            # Niektóre algorytmy mogą wymagać dodatkowych parametrów
            pass

    if not found:
        print("Nie znaleziono pasującego algorytmu.")
        print("\nPróba najpopularniejszych algorytmów:")
        for algo in ["md5", "sha1", "sha224", "sha256", "sha384", "sha512"]:
            h = hashlib.new(algo, input_bytes)
            print(f"  {algo}: {h.hexdigest()}")


if __name__ == "__main__":
    original_text = "R3iSrSNmgU9SFHxVekUD"
    target_hash = "48cab4b54bef42fddaa6353c68a20b369f40026e"

    test_all_hashes(original_text, target_hash)
